# LLMmMm

A simple flask API and static webpage that connects to OpenRouter

# Prerequisites

* docker
* a bash shell
* an ```OPENROUTER_API_KEY``` saved as an environment variable (look it up)

# Running the code

* clone this repository
* ```cd``` to the project directory
* ```bash build.sh```
* ```bash run.sh```
